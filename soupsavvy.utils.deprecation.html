utils.deprecation

Module for deprecation decorators and functions.

deprecated(message: str, warning: ~typing.Type[Warning] = <class 'DeprecationWarning'>) → None[source]

Function for raising warning on deprecated functionalities. It turns off the warning filter for specified warning, raises the warning and then turns it back on to the default.

Parameters

messagestr

Message to be displayed in the warning.

warningType[Warning], optional

Type of warning to be raised, by default DeprecationWarning.

deprecated_class(message: str, warning: ~typing.Type[Warning] = <class 'DeprecationWarning'>) → Callable[source]

Decorator for deprecating classes. It raises a warning when the decorated class is instantiated. Use for classes that will be removed in future versions. Wraps the __init__ method of the class to raise the warning.

Parameters

messagestr

Message to be displayed in the warning.

warningType[Warning], optional

Type of warning to be raised, by default DeprecationWarning.

Returns

Callable

Decorated class.

deprecated_function(message: str, warning: ~typing.Type[Warning] = <class 'DeprecationWarning'>) → Callable[source]

Decorator for deprecating functions. It raises a warning when the decorated function is called. Use for functions that will be removed in future versions. Can be used as a decorator for methods as well.

Parameters

messagestr

Message to be displayed in the warning.

warningType[Warning], optional

Type of warning to be raised, by default DeprecationWarning.

Returns

Callable

Decorated function.
